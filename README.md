# 124250006_AkmaSyafitri_Tugas1

NetFilm merupakan prototype aplikasi berbasis Command Line Interface (CLI) dibuat menggunakan bahasa C++ untuk membantu pengelolaan database film. Program ini dirancang agar pengguna dapat menyimpan, mengurutkan, dan mencari data film.

Fitur:
1. Menyimpan data
2. Pengurutan (Sorting) - Bubble Sort dan Quick Sort
3. Pencarian (Searching) - Linear Search dan Binary Search
